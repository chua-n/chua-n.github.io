                              1. еҢ…з®ЎзҗҶеҹәзЎҖ

                              еңЁ Linux дёҠжңүеҗ„з§ҚеҢ…з®ЎзҗҶзі»з»ҹ(package management system, PMS)пјҢд»ҘеҸҠз”ЁжқҘиҝӣиЎҢиҪҜд»¶е®үиЈ…гҖҒз®ЎзҗҶе’ҢеҲ йҷӨзҡ„е‘Ҫд»ӨиЎҢе·Ҙе…·гҖӮ

                              иҪҜд»¶еҢ…йҖҡеёёдјҡдҫқиө–е…¶д»–зҡ„еҢ…пјҢдёәдәҶиҪҜд»¶иғҪеӨҹжӯЈеёёиҝҗиЎҢпјҢиў«дҫқиө–зҡ„еҢ…еҝ…йЎ»жҸҗеүҚе®үиЈ…еңЁзі»з»ҹдёӯпјҢPMS е·Ҙе…·е°ҶдјҡжЈҖжөӢиҝҷдәӣдҫқиө–е…ізі»пјҢе№¶еңЁе®үиЈ…йңҖиҰҒзҡ„еҢ…д№ӢеүҚе…Ҳе®үиЈ…еҘҪжүҖжңүйўқеӨ–зҡ„иҪҜд»¶еҢ…гҖӮ

                              еҗ„з§Қдё»жөҒ linux еҸ‘иЎҢзүҲйғҪйҮҮз”ЁдәҶжҹҗз§ҚеҪўејҸзҡ„еҢ…з®ЎзҗҶзі»з»ҹжқҘжҺ§еҲ¶иҪҜд»¶е’Ңеә“зҡ„е®үиЈ…пјҢPMS еҲ©з”ЁдёҖдёӘж•°жҚ®еә“жқҘи®°еҪ•еҗ„з§Қзӣёе…іеҶ…е®№пјҡLinux зі»з»ҹдёҠе·Іе®үиЈ…дәҶд»Җд№ҲиҪҜд»¶еҢ…пјӣжҜҸдёӘеҢ…е®үиЈ…дәҶд»Җд№Ҳж–Үд»¶пјӣжҜҸдёӘе·Іе®үиЈ…иҪҜд»¶еҢ…зҡ„зүҲжң¬гҖӮ

                              еҗ„з§Қеҗ„ж ·зҡ„иҪҜд»¶еҢ…йғҪеӯҳеӮЁеңЁжңҚеҠЎеҷЁдёҠпјҢеҸҜд»ҘеҲ©з”Ёжң¬ең° Linux зі»з»ҹдёҠзҡ„ PMS е·Ҙе…·йҖҡиҝҮдә’иҒ”зҪ‘иҝӣиЎҢи®ҝй—®пјҢиҝҷдәӣжңҚеҠЎеҷЁз§°дёәд»“еә“(repository)гҖӮ

                              PMS е·Ҙе…·еҸҠзӣёе…іе‘Ҫд»ӨеңЁдёҚеҗҢзҡ„ Linux еҸ‘иЎҢзүҲдёҠжңүеҫҲеӨ§зҡ„дёҚеҗҢпјҢLinux дёӯе№ҝжіӣдҪҝз”Ёзҡ„дёӨз§Қдё»иҰҒзҡ„ PMS еҹәзЎҖе·Ҙе…·жҳҜ dpkg е’Ң rpmпјҢиҝҷдёӨдёӘе·Ҙе…·жҳҜеҗ„дёӘ Linux еҸ‘иЎҢзүҲдёҠзҡ„ PMS ж ёеҝғпјҢдҪҶе№¶йқһе…ЁйғЁзҡ„ PMSпјҢLinux еҸ‘иЎҢзүҲйғҪжңүеҗ„иҮӘзҡ„еҹәдәҺиҝҷдәӣе‘Ҫд»Өзҡ„зү№е®ҡ PMS е·Ҙе…·гҖӮ

                              е·Ҙе…·дҪҝз”ЁеҜ№иұЎиҜҙжҳҺ
                              dpkgдҪҝз”ЁеҜ№иұЎдёәеҹәдәҺ Debian зҡ„еҸ‘иЎҢзүҲпјҢеҰӮ Ubuntu, Linux Mintdpkg дјҡзӣҙжҺҘе’Ң Linux зі»з»ҹдёҠзҡ„ PMS дәӨдә’пјҢз”ЁжқҘе®үиЈ…гҖҒз®ЎзҗҶгҖҒеҲ йҷӨиҪҜд»¶еҢ…
                              rmpдҪҝз”ЁеҜ№иұЎдёәеҹәдәҺ Red Hat зҡ„еҸ‘иЎҢзүҲпјҢеҰӮ Fedora, openSUSE, Mandrivarmp жҳҜиҝҷдәӣ Linux зі»з»ҹдёҠ PMS зҡ„еә•еұӮеҹәзЎҖпјҢеҸҜе®үиЈ…гҖҒз®ЎзҗҶгҖҒеҲ йҷӨиҪҜд»¶еҢ…
