Java жҸҗдҫӣдәҶ String, StringBuffer, StringBuilder дёүдёӘзұ»жқҘе°ҒиЈ…еӯ—з¬ҰдёІпјҢе№¶жҸҗдҫӣдәҶдёҖзі»еҲ—ж–№жі•гҖӮ

  1. String зұ»жҳҜдёҚеҸҜеҸҳзұ»пјҢдёҖж—ҰдёҖдёӘ String еҜ№иұЎиў«еҲӣе»әпјҢеҢ…еҗ«еңЁиҝҷдёӘеҜ№иұЎдёӯзҡ„еӯ—з¬ҰеәҸеҲ—жҳҜдёҚеҸҜж”№еҸҳзҡ„пјҢзӣҙиҮіиҝҷдёӘеҜ№иұЎиў«й”ҖжҜҒпјӣ

  2. StringBuilder зұ»зҡ„еҜ№иұЎд»ЈиЎЁдёҖдёӘеӯ—з¬ҰеәҸеҲ—еҸҜеҸҳзҡ„еӯ—з¬ҰдёІпјҢеҪ“дёҖдёӘ StringBuilder еҜ№иұЎиў«еҲӣе»әд»ҘеҗҺпјҢйҖҡиҝҮе…¶жҸҗдҫӣзҡ„ append(), insert(), reverse(), setCharAt(), setLength() зӯүж–№жі•еҸҜд»Ҙж”№еҸҳиҝҷдёӘеӯ—з¬ҰдёІеҜ№иұЎзҡ„еӯ—з¬ҰеәҸеҲ—гҖӮдёҖж—ҰйҖҡиҝҮ StringBuilder з”ҹжҲҗдәҶжңҖз»ҲжғіиҰҒзҡ„еӯ—з¬ҰдёІпјҢе°ұеҸҜд»Ҙи°ғз”Ёе®ғзҡ„ toString()ж–№жі•е°Ҷе…¶иҪ¬жҚўдёәдёҖдёӘ String еҜ№иұЎгҖӮ

  3. StringBuffer зұ»еҗҢ StringBuilder зұ»еҮ д№ҺдёҖж ·пјҢе”ҜдёҖзҡ„е·®еҲ«жҳҜпјҢStringBuilder жҳҜзәҝзЁӢдёҚе®үе…Ёзҡ„пјҢиҖҢ StringBuffer жҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢеӣ иҖҢжҖ§иғҪз•ҘдҪҺгҖӮиӢҘдёҚйңҖиҖғиҷ‘зәҝзЁӢе®үе…ЁпјҢйҖҡеёёдјҡдҪҝз”ЁеүҚиҖ…гҖӮ
