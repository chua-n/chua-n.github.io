# 题目

> See [HERE](https://leetcode.com/problems/same-tree/).

<div><p>Given two binary trees, write a function to check if they are the same or not.</p>

<p>Two binary trees are considered the same if they are structurally identical and the nodes have the same value.</p>

<p><strong>Example 1:</strong></p>

<pre><strong>Input:</strong>     1         1
          / \       / \
         2   3     2   3

        [1,2,3],   [1,2,3]

<strong>Output:</strong> true
</pre>

<p><strong>Example 2:</strong></p>

<pre><strong>Input:</strong>     1         1
          /           \
         2             2

        [1,2],     [1,null,2]

<strong>Output:</strong> false
</pre>

<p><strong>Example 3:</strong></p>

<pre><strong>Input:</strong>     1         1
          / \       / \
         2   1     1   2

        [1,2,1],   [1,1,2]

<strong>Output:</strong> false
</pre>
</div>

# Code

## 1. C++

> Cost: 4 ms (58.46%), 9.9 MB (41.67%)

```C++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        queue<TreeNode *> q1, q2;
        q1.push(p); q2.push(q);
        while(!q1.empty() && !q2.empty()){
            auto cur1 = q1.front(), cur2 = q2.front();
            q1.pop(); q2.pop();
            if (cur1 && cur2){
                if (cur1->val != cur2->val) return false;
                q1.push(cur1->left); q1.push(cur1->right);
                q2.push(cur2->left); q2.push(cur2->right);
            }
            else if(!cur1 && !cur2) continue;
            else return false;
        }
        return true;
    }
};
```

## 2. Python

> Cost: 36 ms (43.90%), 13.1 MB ()

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def isSameTree(self, p: TreeNode, q: TreeNode) -> bool:
        if not p or not q:
            return p == q
        if p.val == q.val:
            return self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)
        else:
            return False
```
