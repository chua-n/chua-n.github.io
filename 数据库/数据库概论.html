                    еңЁе…ізі»ж•°жҚ®еә“дёӯпјҢе…ізі»жҳҜйҖҡиҝҮдё»й”®е’ҢеӨ–й”®жқҘз»ҙжҠӨзҡ„гҖӮжҲ‘д»¬еңЁеҗҺйқўдјҡеҲҶеҲ«ж·ұе…Ҙи®Іи§ЈгҖӮ

                    2.3 дёҺExcelиЎЁзҡ„дёҚеҗҢ

                    е’ҢExcelиЎЁжңүжүҖдёҚеҗҢзҡ„жҳҜпјҢе…ізі»ж•°жҚ®еә“зҡ„иЎЁе’ҢиЎЁд№Ӣй—ҙйңҖиҰҒе»әз«ӢдёҖеҜ№еӨҡгҖҒеӨҡеҜ№дёҖе’ҢдёҖеҜ№дёҖзҡ„е…ізі»пјҢиҝҷж ·жүҚиғҪеӨҹжҢүз…§еә”з”ЁзЁӢеәҸзҡ„йҖ»иҫ‘жқҘз»„з»Үе’ҢеӯҳеӮЁж•°жҚ®гҖӮдҫӢеҰӮпјҢдёҖдёӘзҸӯзә§иЎЁпјҡ

                    IDеҗҚз§°зҸӯдё»д»»
                    201дәҢе№ҙзә§дёҖзҸӯзҺӢиҖҒеёҲ
                    202дәҢе№ҙзә§дәҢзҸӯжқҺиҖҒеёҲ

                    жҜҸдёҖиЎҢеҜ№еә”зқҖдёҖдёӘзҸӯзә§пјҢиҖҢдёҖдёӘзҸӯзә§еҜ№еә”зқҖеӨҡдёӘеӯҰз”ҹпјҢжүҖд»ҘзҸӯзә§иЎЁе’ҢеӯҰз”ҹиЎЁзҡ„е…ізі»е°ұжҳҜвҖңдёҖеҜ№еӨҡвҖқпјҡ

                    IDе§“еҗҚзҸӯзә§IDжҖ§еҲ«е№ҙйҫ„
                    1е°ҸжҳҺ201M9
                    2е°Ҹзәў202F8
                    3е°ҸеҶӣ202M8
                    4е°ҸзҷҪ201F9

                    еҸҚиҝҮжқҘпјҢеҰӮжһңжҲ‘д»¬е…ҲеңЁеӯҰз”ҹиЎЁдёӯе®ҡдҪҚдәҶдёҖиЎҢи®°еҪ•пјҢдҫӢеҰӮID=1зҡ„е°ҸжҳҺпјҢиҰҒзЎ®е®ҡд»–зҡ„зҸӯзә§пјҢеҸӘйңҖиҰҒж №жҚ®д»–зҡ„вҖңзҸӯзә§IDвҖқеҜ№еә”зҡ„еҖј201жүҫеҲ°зҸӯзә§иЎЁдёӯID=201зҡ„и®°еҪ•пјҢеҚідәҢе№ҙзә§дёҖзҸӯгҖӮжүҖд»ҘпјҢеӯҰз”ҹиЎЁе’ҢзҸӯзә§иЎЁжҳҜвҖңеӨҡеҜ№дёҖвҖқзҡ„е…ізі»гҖӮ

                    еҰӮжһңжҲ‘д»¬жҠҠзҸӯзә§иЎЁеҲҶжӢҶеҫ—з»ҶдёҖзӮ№пјҢдҫӢеҰӮпјҢеҚ•зӢ¬еҲӣе»әдёҖдёӘж•ҷеёҲиЎЁпјҡ

                    IDеҗҚз§°е№ҙйҫ„
                    A1зҺӢиҖҒеёҲ26
                    A2еј иҖҒеёҲ39
                    A3жқҺиҖҒеёҲ32
                    A4иөөиҖҒеёҲ27

                    зҸӯзә§иЎЁеҸӘеӯҳеӮЁж•ҷеёҲIDпјҡ

                    IDеҗҚз§°зҸӯдё»д»»ID
                    201дәҢе№ҙзә§дёҖзҸӯA1
                    202дәҢе№ҙзә§дәҢзҸӯA3

                    иҝҷж ·пјҢдёҖдёӘзҸӯзә§жҖ»жҳҜеҜ№еә”дёҖдёӘж•ҷеёҲпјҢзҸӯзә§иЎЁе’Ңж•ҷеёҲиЎЁе°ұжҳҜвҖңдёҖеҜ№дёҖвҖқе…ізі»гҖӮ

                    3. SQL

                    з»“жһ„еҢ–жҹҘиҜўиҜӯиЁҖSQL(Structured Query Language)пјҢеҸ‘йҹіsequelжҲ–еӯ—жҜҚS-Q-LпјҢжҳҜдёҖз§Қдё“й—Ёз”ЁжқҘдёҺж•°жҚ®еә“йҖҡдҝЎзҡ„иҜӯиЁҖгҖӮSQLиҜӯеҸҘж—ўеҸҜд»ҘжҹҘиҜўж•°жҚ®еә“дёӯзҡ„ж•°жҚ®пјҢд№ҹеҸҜд»Ҙж·»еҠ гҖҒжӣҙж–°е’ҢеҲ йҷӨж•°жҚ®еә“дёӯзҡ„ж•°жҚ®пјҢиҝҳеҸҜд»ҘеҜ№ж•°жҚ®еә“иҝӣиЎҢз®ЎзҗҶе’Ңз»ҙжҠӨж“ҚдҪңгҖӮ

                    • иҷҪз„¶SQLе·Із»Ҹиў«ANSIз»„з»Үе®ҡд№үдёәж ҮеҮҶпјҢдёҚе№ёең°жҳҜпјҢеҗ„дёӘдёҚеҗҢзҡ„ж•°жҚ®еә“еҜ№ж ҮеҮҶзҡ„SQLж”ҜжҢҒдёҚеӨӘдёҖиҮҙгҖӮе№¶дё”пјҢеӨ§йғЁеҲҶж•°жҚ®еә“йғҪеңЁж ҮеҮҶзҡ„SQLдёҠеҒҡдәҶжү©еұ•гҖӮд№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңеҸӘдҪҝз”Ёж ҮеҮҶSQLпјҢзҗҶи®әдёҠжүҖжңүж•°жҚ®еә“йғҪеҸҜд»Ҙж”ҜжҢҒпјҢдҪҶеҰӮжһңдҪҝз”ЁжҹҗдёӘзү№е®ҡж•°жҚ®еә“зҡ„жү©еұ•SQLпјҢжҚўдёҖдёӘж•°жҚ®еә“е°ұдёҚиғҪжү§иЎҢдәҶгҖӮдҫӢеҰӮпјҢOracleжҠҠиҮӘе·ұжү©еұ•зҡ„SQLз§°дёәPL/SQLпјҢMicrosoftжҠҠиҮӘе·ұжү©еұ•зҡ„SQLз§°дёәT-SQLгҖӮ

                    • зҺ°е®һжғ…еҶөжҳҜпјҢеҰӮжһңжҲ‘д»¬еҸӘдҪҝз”Ёж ҮеҮҶSQLзҡ„ж ёеҝғеҠҹиғҪпјҢйӮЈд№ҲжүҖжңүж•°жҚ®еә“йҖҡеёёйғҪеҸҜд»Ҙжү§иЎҢгҖӮдёҚеёёз”Ёзҡ„SQLеҠҹиғҪпјҢдёҚеҗҢзҡ„ж•°жҚ®еә“ж”ҜжҢҒзҡ„зЁӢеәҰйғҪдёҚдёҖж ·гҖӮиҖҢеҗ„дёӘж•°жҚ®еә“ж”ҜжҢҒзҡ„еҗ„иҮӘжү©еұ•зҡ„еҠҹиғҪпјҢйҖҡеёёжҲ‘д»¬жҠҠе®ғд»¬з§°д№ӢдёәвҖңж–№иЁҖвҖқгҖӮ

                    жҖ»зҡ„жқҘиҜҙпјҢSQLиҜӯиЁҖеҸҜеҲҶдёәеҰӮдёӢеҮ з§Қзұ»еһӢпјҡ

                    SQLиҜӯеҸҘзҡ„зұ»еһӢиҜҙжҳҺ
                    DDL(Data Definition Language)DDLз”ЁжқҘж“ҚдҪңж•°жҚ®еә“гҖҒиЎЁз»“жһ„гҖӮйҖҡеёёпјҢDDLз”ұж•°жҚ®еә“з®ЎзҗҶе‘ҳжү§иЎҢгҖӮ
                    DML(Data Manipulation Language)DMLдёәз”ЁжҲ·жҸҗдҫӣж·»еҠ гҖҒеҲ йҷӨгҖҒжӣҙж–°ж•°жҚ®зҡ„иғҪеҠӣпјҢиҝҷдәӣжҳҜеә”з”ЁзЁӢеәҸеҜ№ж•°жҚ®еә“зҡ„ж—Ҙеёёж“ҚдҪңгҖӮ
                    DQL(Data Query Language)DQLе…Ғи®ёз”ЁжҲ·жҹҘиҜўж•°жҚ®пјҢиҝҷд№ҹжҳҜйҖҡеёёжңҖйў‘з№Ғзҡ„ж•°жҚ®еә“ж—Ҙеёёж“ҚдҪңгҖӮ

                    з®ҖеҚ•ең°иҜҙпјҢSQLе°ұжҳҜи®ҝй—®е’ҢеӨ„зҗҶе…ізі»ж•°жҚ®еә“зҡ„и®Ўз®—жңәж ҮеҮҶиҜӯиЁҖгҖӮд№ҹе°ұжҳҜиҜҙпјҢж— и®әз”Ёд»Җд№Ҳзј–зЁӢиҜӯиЁҖпјҲJavaгҖҒPythonгҖҒC++вҖҰпјүзј–еҶҷзЁӢеәҸпјҢеҸӘиҰҒж¶үеҸҠеҲ°ж“ҚдҪңе…ізі»ж•°жҚ®еә“пјҢжҜ”еҰӮпјҢдёҖдёӘз”өе•ҶзҪ‘з«ҷйңҖиҰҒжҠҠз”ЁжҲ·е’Ңе•Ҷе“ҒдҝЎжҒҜеӯҳе…Ҙж•°жҚ®еә“пјҢжҲ–иҖ…дёҖдёӘжүӢжңәжёёжҲҸйңҖиҰҒжҠҠз”ЁжҲ·зҡ„йҒ“е…·гҖҒйҖҡе…ідҝЎжҒҜеӯҳе…Ҙж•°жҚ®еә“пјҢйғҪеҝ…йЎ»йҖҡиҝҮSQLжқҘе®ҢжҲҗгҖӮ

                    жүҖд»ҘпјҢзҺ°д»ЈзЁӢеәҸзҰ»дёҚејҖе…ізі»ж•°жҚ®еә“пјҢиҰҒдҪҝз”Ёе…ізі»ж•°жҚ®еә“е°ұеҝ…йЎ»жҺҢжҸЎSQLгҖӮ

                    4. NoSQL

                    дҪ еҸҜиғҪиҝҳеҗ¬иҜҙиҝҮNoSQLж•°жҚ®еә“пјҢд№ҹе°ұжҳҜйқһSQLзҡ„ж•°жҚ®еә“пјҢеҢ…жӢ¬MongoDBгҖҒCassandraгҖҒDynamoзӯүзӯүпјҢе®ғд»¬йғҪдёҚжҳҜе…ізі»ж•°жҚ®еә“гҖӮ

                    жңүеҫҲеӨҡдәәйј“еҗ№зҺ°д»ЈWebзЁӢеәҸе·Із»Ҹж— йңҖе…ізі»ж•°жҚ®еә“дәҶпјҢеҸӘйңҖиҰҒдҪҝз”ЁNoSQLе°ұеҸҜд»ҘпјҢдҪҶдәӢе®һдёҠпјҢSQLж•°жҚ®еә“д»Һе§ӢиҮіз»Ҳд»ҺжңӘиў«еҸ–д»ЈиҝҮгҖӮеӣһйЎҫдёҖдёӢNoSQLзҡ„еҸ‘еұ•еҺҶзЁӢпјҡ

                    • 1970: NoSQL = We have no SQL
                    • 1980: NoSQL = Know SQL
                    • 2000: NoSQL = No SQL!
                    • 2005: NoSQL = Not only SQL
                    • 2013: NoSQL = No, SQL!

                    д»ҠеӨ©пјҢSQLж•°жҚ®еә“д»Қз„¶жүҝжӢ…дәҶеҗ„з§Қеә”з”ЁзЁӢеәҸзҡ„ж ёеҝғж•°жҚ®еӯҳеӮЁпјҢиҖҢNoSQLж•°жҚ®еә“дҪңдёәSQLж•°жҚ®еә“зҡ„иЎҘе……пјҢдёӨиҖ…дёҚеҶҚжҳҜдәҢйҖүдёҖзҡ„й—®йўҳпјҢиҖҢжҳҜдё»д»Һе…ізі»гҖӮ

                    5. жӮІи§Ӯй”ҒдёҺд№җи§Ӯй”Ғ

                    жӮІи§Ӯй”Ғ(Pessimistic Lock) пјҢйЎҫеҗҚжҖқд№үпјҢе°ұжҳҜеҫҲжӮІи§ӮпјҢжҜҸж¬ЎеҺ»жӢҝж•°жҚ®зҡ„ж—¶еҖҷйғҪи®ӨдёәеҲ«дәәдјҡдҝ®ж”№пјҢжүҖд»ҘжҜҸж¬ЎеңЁжӢҝж•°жҚ®зҡ„ж—¶еҖҷйғҪдјҡдёҠй”ҒпјҢиҝҷж ·еҲ«дәәжғіжӢҝиҝҷдёӘж•°жҚ®ж—¶е°ұдјҡйҳ»еЎһпјҢзӣҙеҲ°е®ғжӢҝеҲ°й”ҒгҖӮдј з»ҹзҡ„е…ізі»еһӢж•°жҚ®еә“йҮҢиҫ№е°ұз”ЁеҲ°дәҶеҫҲеӨҡиҝҷз§Қй”ҒжңәеҲ¶пјҢжҜ”еҰӮиЎҢй”ҒгҖҒиЎЁй”ҒгҖҒиҜ»й”ҒгҖҒеҶҷй”ҒзӯүпјҢйғҪжҳҜеңЁеҒҡж“ҚдҪңд№ӢеүҚе…ҲдёҠй”ҒгҖӮ

                    д№җи§Ӯй”Ғ(Optimistic Lock) пјҢйЎҫеҗҚжҖқд№үпјҢе°ұжҳҜеҫҲд№җи§ӮпјҢжҜҸж¬ЎеҺ»жӢҝж•°жҚ®зҡ„ж—¶еҖҷйғҪи®ӨдёәеҲ«дәәдёҚдјҡдҝ®ж”№пјҢжүҖд»ҘдёҚдјҡдёҠй”ҒпјҢдҪҶжҳҜеңЁжӣҙж–°ж•°жҚ®зҡ„ж—¶еҖҷдјҡеҲӨж–ӯдёҖдёӢеңЁжӯӨжңҹй—ҙеҲ«дәәжңүжІЎжңүеҺ»жӣҙж–°иҝҷдёӘж•°жҚ®пјҢеҲӨж–ӯзҡ„дҫқжҚ®жҳҜдҪҝз”ЁзүҲжң¬еҸ·зӯүжңәеҲ¶гҖӮд№җи§Ӯй”ҒйҖӮз”ЁдәҺеӨҡиҜ»зҡ„еә”з”Ёзұ»еһӢпјҢиҝҷж ·еҸҜд»ҘжҸҗй«ҳеҗһеҗҗйҮҸгҖӮRedisе°ұжҳҜеҲ©з”Ёиҝҷз§Қ check-and-set зҡ„д№җи§Ӯй”ҒжңәеҲ¶е®һзҺ°дәӢеҠЎзҡ„гҖӮ
